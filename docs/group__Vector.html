Vector contructor.

Vector constructor function that initializes vector with properties packed in opts struct. Space for initial_cap elements will be reserved. In case of allocation fail null pointer will be returned via vector argument.

Parameters

Returns
Fresh new instance of the vector or NULL if allocation failed.

Provides a location where user can put a header for the derived class.

Function returns a pointer to reserved space after vector's control struct. Space for the header extension has to be preallocated on vector creation, size of this region is specified by vector_t::data_offset property in vector_opts_t struct.

Shift range of elements.

Shifting length elements at offset by shift times in direction of a sign. Data will be overriden by shifted range.
